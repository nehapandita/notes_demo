1. Install NodeJS from the following URL for your machine.

https://nodejs.org/download

2. Place this folder in your machine somewhere

3. Open "Node.js Command Prompt" from Start Menu, CD to the "Notes" directory e.g. C:\>CD Users/User/Desktop/Notes and then type following command and hit enter to install all the packages:

npm install

4. Then type following command and hit enter to run the app:

node app.js

5. Open your browser, and type following url:

http://localhost:3000

6. You should be able to go straight to a note if specified in the url ( notes.com/4 ).
